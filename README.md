# PDF Resume Builder

A small client-side project to build and export PDF resumes.

Files
- `index.html` — main UI
- `script.js` — project logic
- `style.css` — styles

Usage
1. Open `index.html` in your browser.
2. Fill in your resume details in the form.
3. Use the provided button to generate and download the PDF.

Development
- No build step required — static HTML/CSS/JS.
- For local testing, you can run a simple HTTP server, e.g.:

  ```bash
  python -m http.server 8000
  ```
